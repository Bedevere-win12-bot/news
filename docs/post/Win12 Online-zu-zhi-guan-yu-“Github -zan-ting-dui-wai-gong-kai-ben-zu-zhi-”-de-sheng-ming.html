Win12 Online组织关于“Github 暂停对外公开本组织”的声明

Github是全球最大的源代码托管平台，其盛名源自它提供优质的服务——允许开发者自由创建仓库、存储代码。 许多知名库与开发框架都采用GitHub作为代码托管平台，可以说 Github 对全球开源社区的发展起到举足轻重的作用。本组织也早在 2022 年就将 Win12 项目的代码开源并托管至 Github，并尽我们最大努力对开源社区的发展添砖加瓦。同时我们通过 Github Pages 服务向全球用户提供本项目的在线体验网页。可以说没有 Github ，也没有今天的我们和充满活力、蓬勃发展的 Win12 项目社区。

Win12 Online 组织自成立以来，就致力于整合 Win12 项目（以下简称“本项目”）的资源。为使身处中国大陆的用户能更为便捷的访问本项目，体会到本项目当中的乐趣，我们向南京大学开源软件镜像站等一系列高校镜像站申请镜像本项目的桌面发行版，并得到了南京大学的老师和同学的支持，使使用桌面发行版的用户的下载速度因此大幅提升。与此同时，我们的贡献者也一直保持着较高的频率向本项目贡献代码，不断为本项目社区的发展注入新活力、提供新动能。

我们必须指出，本组织一直秉承着开源社区的自由开放的精神，也从未做出过任何明确违反《Github 服务条款》（GitHub Terms of Service）的行为。但 Github 却在 2026 年 6 月 25 日以莫须有的理由，无端将本组织标记并隐藏。可以说这一决定是 Github 未经明确的考量而毫无根据地作出的，是彻彻底底的欲加之罪，是完完全全的黑箱操作。

我们不是第一个被 Github 如此对待的组织，在 Github 的社区页面 当中，若你键入“organization flagged”，你将发现有相当多的组织都被 Github 错误地标记且申诉无果。这也不是 Github 第一次做出诸如此类的对开发者的限制，早在 2019 和 2021 年 Github 就曾陷入过关于“制裁伊朗等国的开发者”和“国会山骚乱评论”的争议。

我们必须指出，GitHub 此次对本组织进行标记的行为是完全错误的，在 Github 社区范围内对其他的组织的标记也是完全不可理喻的。这种对于开发者的大规模“封杀”，会极大损害开发者的信心，动摇开发者对 GitHub 的信任。我们对此极为痛心，并强烈谴责。与此同时，我们已经向 Github 提交了申诉，无论申诉的结果如何，这种不公的对待，使我们已对 Github 失去了信心，我们或决定将代码仓库迁移至 Codeberg 或其他自托管平台。

感谢所有 Win12 项目的用户一路真心相伴！
